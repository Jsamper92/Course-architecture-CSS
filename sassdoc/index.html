    set-keyframes

    @mixin set-keyframes($animations) { ... }

    Description

    set-keyframes - Mixin que crea todas las animaciones genericas

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $animations

    Recibe como parametro un mapa con todas las animaciones genericas

    		Map— none

    Example

    @include set-keyframes($animations);

    Requires

    • [variable] animations

    variables

    animations

    $animations: (
  animation-1:(
    0:(
      opacity:0,
      transform:translateY(-100px)
    ),
    50:(
       opacity:0.5,
       transform:translateY(-50px)
    ),
    100:(
      opacity:1,
      transform:translateY(0px)
    )
  ),
  animation-2:(
    0:(
      transform:scale(0)
    ),
    100:(
      transform:scale(1)
    )
  )
);

    Description

    Mapa donde definimos todas las animaciones genericas

    Type

    Map

    Example

    @keyframes animation-1 {
  0% {
    opacity: 0;
    transform: translateY(-100px);
  }
  50% {
    opacity: 0.5;
    transform: translateY(-50px);
  }
  100% {
    opacity: 1;
    transform: translateY(0px);
  }
}
@keyframes animation-2 {
  0% {
    transform: scale(0);
  }
  100% {
    transform: scale(1);
  }
}

    Used by

    • [mixin] set-keyframes

    Description

    Mixin que crea una fuente de texto

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    Nombre de la fuente de texto

    		Any— none
    $file

    Nombre del archivo de la fuente

    		Any— none
    $weight

    Grosor de la fuente

    		Any500

    Example

  font-family: font-regular;
  font-display: auto;
  src: url("/sass/fonts/barlow-regular.ttf");
  font-style: normal;
  font-weight: 500;
}

    icons

    $icons: (
  first-page: "\e92d",
  last-page: "\e92e",
  button-success: "\e92c",
  table-options: "\e92b",
  mouse-square: "\e928",
  mouse-circle: "\e929",
  responsive-table: "\e92a",
  arrow-down: "\e916"
);

    Description

    Mapa donde se alojan todos los iconos del proyecto desde una fuente icónica, de esta manera cada vez que queramos añadir un nuevo icono, simplemente tendremos que importar la nueva fuente, y el nombre del icono con su respectivo valor.

    Type

    Map

    Example

    .icon-first-page::before {
   content: "\e92d";
}
